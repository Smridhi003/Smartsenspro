# SmartSense Pro

Industrial IoT Predictive Maintenance Dashboard built with React.

## Features

- Industrial Sensor Dashboard
- Machine Health Index (MHI)
- Predictive Maintenance
- Z-Score Anomaly Detection
- Exponential Moving Average (EMA)
- AI Diagnostics (Demo Mode)
- Interactive Charts

## Technologies

- React
- JavaScript
- Recharts

## Installation

```bash
npm install
npm run dev
```
